# ml-regression-project
# Car Price Prediction using Linear Regression

## 📌 Project Overview

This project is based on Machine Learning using Linear Regression to predict car prices from automobile dataset features such as engine size, horsepower, fuel type, body style, and other car specifications.

The project covers the complete Machine Learning workflow including:

* Data Cleaning
* Handling Missing Values
* Data Preprocessing
* Encoding Categorical Data
* Exploratory Data Analysis (EDA)
* Model Training
* Prediction and Evaluation

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Seaborn
* Scikit-learn
* Jupyter Notebook

---

Some important dataset columns used:

* symboling

* body-style
* drive-wheels
* engine-type
* horsepower
* peak-rpm
* price

---

## ⚙️ Steps Performed

1. Imported required libraries
2. Loaded and explored dataset
3. Replaced missing values (`?`) with `NaN`
4. Cleaned and transformed data
5. Encoded categorical columns using:

   * Label Encoding
   * One Hot Encoding
6. Split dataset into training and testing data
7. Trained Linear Regression model
8. Predicted car prices
9. Evaluated model performance using:

   * MAE
   * MSE
   * R² Score

---

## 📊 Machine Learning Model

Linear Regression was used to build the prediction model.

Linear Regression equation:

y = mx + b

Where:

* y = predicted output
* x = input feature
* m = slope
* b = intercept
